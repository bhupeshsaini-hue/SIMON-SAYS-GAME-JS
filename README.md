# SIMON-SAYS-GAME-JS
The Simon Says Game is a fun and interactive web-based memory game built using HTML, CSS, and JavaScript. The game challenges players to remember and repeat a sequence of colors generated randomly by the system. With each level, the sequence becomes longer, increasing the difficulty and testing the player's memory and concentration skills.
# Features
1.Random color sequence generation
2.Memory-based gameplay
3.Level progression system
4.Game over detection on wrong input
5.Interactive UI with visual effects
# How to Play
1.Press any key to start the game
2.Watch the sequence of colors carefully
3.Repeat the sequence by clicking the buttons in the same order
4.Progress to the next level if correct
5.Game ends if you make a mistake
